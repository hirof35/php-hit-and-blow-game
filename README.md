# Hit & Blow (PHP Web Edition)

PHPのセッション機能を利用した、ブラウザ上で遊べる数字当てゲーム「ヒットアンドブロー（Hit & Blow）」です。
3桁から5桁までの難易度選択に対応しており、直感的なUIで過去の推測履歴を確認しながらプレイできます。

## 🎮 ゲームルール

あらかじめシステム側がランダムに用意した「重複のない数字（0〜9）」を当てるゲームです。
数字を予想して送信すると、以下のヒントが返されます。

- **Hit (H):** 数字も配置されている位置も合っている。
- **Blow (B):** 数字は合っているが、配置されている位置が違う。

（例：正解が「1234」で、予想が「1489」の場合 ＝ `1 Hit / 1 Blow`）
設定された桁数すべての数字と位置を完全に当てることができればゲームクリアです！

## ✨ 主な機能

- **難易度設定:** 3桁 (Easy) / 4桁 (Normal) / 5桁 (Hard) の3段階から選択可能。
- **ゲーム状態の維持:** ページ遷移やリロードを行っても、PHPのセッション（`$_SESSION`）により、正解の数字や現在のターン数、推測履歴が保持されます。
- **リアルタイム履歴表示:** 過去に入力した数字と、そのときの結果（Hit / Blow）が最新順にテーブル表示されます。
- **安全な入力チェック:** 桁数不足、数字以外の入力、数字の重複を検知し、適切なエラーメッセージを表示します。
- **セキュリティー:** `htmlspecialchars` によるXSS（クロスサイトスクリプティング）対策済み。

## 🛠️ 動作環境

- PHP 7.4 以上 （PHP 8.x 推奨）
- Webブラウザ（Google Chrome, Safari, Microsoft Edge 等）

## 🚀 導入・実行方法

### 1. ファイルの配置
リポジトリ内の `index.php` を、お使いのWebサーバーの公開ディレクトリ（`htdocs` や `www` など）に配置してください。

### 2. ローカルサーバーでのクイック起動
PHPがインストールされている環境であれば、ターミナル（コマンドプロンプト）を開き、ファイルがあるディレクトリで以下のコマンドを実行するだけで簡単に起動できます。

```bash
php -S localhost:8000
起動後、ブラウザで http://localhost:8000 にアクセスしてください。
